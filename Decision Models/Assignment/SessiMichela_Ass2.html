Predicting the letter B

To warm up, start by predicting whether or not the letter is B. First, create a new variable called IsB in your dataset, which takes value Yes if the letter is B, and No if the letter is not B. Then randomly split your dataset into a training set and a testing set, putting 50% of the data in each set.

Letters$Isb <- ifelse(Letters$Letter=="B", "yes", "no")
Letters$Isb <- as.factor(Letters$Isb)
set.seed(123) # to get repeatable results.
spl = sample.split(Letters$Isb, SplitRatio = 0.5)
Training = subset(Letters, spl==TRUE)
Testing = subset(Letters, spl==FALSE)

Remember to not use the variable Letter as one of the independent variables in the model, as this is related to what we are trying to predict! Select reasonable parameter values for the model, and justify your parameter choices. What is the accuracy of this CART model on the test set?
FormulaCART = Isb ~ Xbox +Ybox + Width + Height + Onpix + Xbar + Ybar + X2bar + Y2bar + XYbar + X2Ybar + XY2bar + Xedge + XedgeYcor + Yedge + YedgeXcor

Let start creating a for cycle to choose the better minbucket.

seq = seq( from = 1, to = 30, by = 1)
accuracy<- c()
for (n in seq) { 
  DT = rpart(formula = FormulaCART, method="class", data=Training, 
              control=rpart.control(minbucket= n))
  predictedDecisions <- predict(DT, Testing, type="class")
  cm <- confusionMatrix(predictedDecisions, Testing$Isb)
  accuracy[n] <- cm$overall[1]}
plot(accuracy, xlab = "minbucket", col= 'blue', main= "For cycle to choose minbucket parameter", col.main='blue')

Set the minbucket parameter = 6. In this way we have the better performance with minimum work. For cycle to choose the better maxdepth:
